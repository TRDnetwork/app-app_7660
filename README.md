This app is a static personal portfolio with no user data storage or authentication requirements. As such, no Supabase tables, RLS policies, storage buckets, or edge functions are needed. The contact form is handled entirely by a Vercel serverless function (`api/send-email.ts`) owned by the Email Agent, which integrates with Resend for email delivery. Supabase is reserved for future enhancements (e.g., project analytics, user messaging, or CMS) but plays no active role in the current implementation. Features like responsive layout, form validation, and toast notifications are client-side only.
